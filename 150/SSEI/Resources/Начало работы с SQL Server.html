Начало работы с SQL Server

Благодарим за установку Microsoft SQL Server. Если вы подключаетесь к SQL Server с удаленного компьютера, выполните необходимые требования, упомянутые в файле "Перед началом работы", который находится в папке Ресурсы.

Примеры

Примеры подключения к базе данных SQL Server, примеры разработки с использованием разных функций, таких как выполняющаяся в памяти OLTP, службы R и многие другие, см. в разделе Официальный репозиторий Microsoft GitHub, содержащий примеры кода для SQL Server.

Также можно воспользоваться репозиторием с примерами Базы данных SQL Azure и эталонными реализациями, чтобы ознакомиться с различными примерами эталонных реализаций для C#, java, node.js, php, python и т. д. При работе с этими примерами потребуется соответствующим образом изменить имя сервера в строке подключения.

Форум поддержки

Если у вас есть вопросы по любой из перечисленных выше тем, можете разместить их на форуме MSDN: Начало работы с SQL Server или в Twitter с хэштегом #sqlhelp и тегом @SQLServer. В MSDN есть и более специализированные форумы, например форум Transact-SQL, который может помочь с написанием кода Transact-SQL для запроса базы данных, или форум по документации SQL Server.

Вы можете публиковать вопросы и предложения в UserVoice: SQL Server.
